#  AI Underwater Monitoring System

An AI-based system for detecting marine life and underwater objects using YOLOv8 and CLAHE enhancement.

##  Features
- Image & Video Detection
- Live Camera Monitoring
- CLAHE Image Enhancement
- Fish & Garbage Detection
- Detection Statistics Dashboard

##  Technologies Used
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- Streamlit

##  How to Run
bash
pip install -r requirements.txt
streamlit run app.py
